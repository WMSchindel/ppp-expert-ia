# Capítulo 1

# Bem-vindo ao PPP Expert IA

---

## Objetivo deste livro

Este livro documenta integralmente o desenvolvimento do projeto
PPP Expert IA.

Seu objetivo não é apenas mostrar o código-fonte da aplicação,
mas explicar cada decisão arquitetural, cada conceito utilizado
e o funcionamento interno das tecnologias empregadas.

Ao final da leitura, o leitor deverá ser capaz de compreender,
manter e evoluir o sistema de forma independente.

---

## O que será desenvolvido

Durante este livro construiremos um produto comercial denominado
PPP Expert IA.

O sistema será capaz de interpretar documentos relacionados à
Segurança e Saúde do Trabalho (SST), consolidar informações,
validar inconsistências e gerar automaticamente o Perfil
Profissiográfico Previdenciário (PPP).

O PPP será apenas o primeiro módulo de uma plataforma maior,
denominada SST Platform.

---

## Filosofia

O projeto segue um princípio simples.

Primeiro compreender.

Depois projetar.

Depois implementar.

Nunca o contrário.

---

## Metodologia

Cada funcionalidade seguirá obrigatoriamente as etapas abaixo.

1. Levantamento do requisito.

2. Projeto.

3. Arquitetura.

4. Implementação.

5. Testes.

6. Documentação.

7. Registro da decisão arquitetural.

8. Atualização do livro.

9. Commit.

---

## Organização do projeto

O software foi organizado utilizando princípios da Clean
Architecture e do SOLID.

Cada módulo possui uma única responsabilidade.

Nenhuma decisão arquitetural é tomada sem justificativa técnica.

---

## Como utilizar este livro

Este livro deve ser lido sequencialmente.

Cada capítulo depende do anterior.

Os exemplos apresentados são exatamente os mesmos utilizados no
desenvolvimento do software.

Não existem exemplos fictícios.

Todo código mostrado faz parte do produto final.
